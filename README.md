# 내 Jekyll 블로그

이 저장소는 GitHub Pages와 Jekyll을 사용하여 만든 개인 블로그입니다.

## 로컬에서 실행하기

1. Ruby와 Jekyll 설치
2. 저장소 클론
   ```
   git clone https://github.com/사용자이름/저장소이름.git
   cd 저장소이름
   ```
3. 의존성 설치
   ```
   bundle install
   ```
4. 서버 실행
   ```
   bundle exec jekyll serve
   ```
5. 브라우저에서 `http://localhost:4000` 접속
